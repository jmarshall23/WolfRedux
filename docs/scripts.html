Return to Castle Wolfenstein Script Actions Documentation

Movement Actions

gotomarker: Moves the character to a specified marker at the default movement speed.

runtomarker: Moves the character to a specified marker at a running speed.

walktomarker: Moves the character to a specified marker at a walking speed.

crouchtomarker: Moves the character to a specified marker while crouching.

Interaction Actions

gotocast: Moves the character to a specified cast (NPC).

runtocast: Runs the character to a specified cast.

walktocast: Walks the character to a specified cast.

crouchtocast: Crouches the character to a specified cast.

followcast: Makes the character follow a specified cast.

Sound and Animation Actions

playsound: Plays a specified sound.

playanim: Plays a specified animation.

clearanim: Clears the current animation.

Control Actions

wait: Pauses the script for a specified amount of time.

abort_if_loadgame: Aborts the script if the game is loaded from a save.

trigger: Triggers another script or event.

resetscript: Resets the script to its initial state.

Combat and Equipment Actions

setammo: Sets the amount of ammo for a specified weapon.

givearmor: Gives armor to the character.

giveweapon: Gives a specified weapon to the character.

Utility Actions

alertentity: Triggers an entity's script.

savegame: Saves the game.

godmode: Toggles god mode for the character.

Player Interaction Actions

lockplayer: Locks player controls, preventing any player action.

unlockplayer: Unlocks player controls, allowing player actions again. (Implied action not listed previously)

Mission Control Actions

missionfailed: Triggers a mission failed state.

missionsuccess, objectivemet: Triggers a mission success state.

objectivesneeded: Specifies the number of objectives needed to complete the mission.

Advanced Actions

mu_start, mu_play, mu_stop, mu_fade, mu_queue: Control music playback and fading.

teleport: Teleports the character to a specified location.

changelevel: Changes the current level to a specified one.

endgame: Triggers the end of the game sequence.

foundsecret: Marks a secret as found

lightningdamage: Applies lightning damage to the entity or player.

Environmental and Scenario Actions

teleport: Teleports the character to a specified location, instantly changing their position in the game world.

cigarette: Initiates the cigarette animation or effect for a character. (Specific to thematic actions)

Condition and State Actions

noaidamage: Toggles whether AI can receive damage, enabling or disabling their vulnerability.

nosight, sight: Toggles the AI's ability to see, affecting their reaction to the player and other entities.

noavoid, avoid: Toggles the AI's avoidance behavior, influencing how they navigate around obstacles or dangerous areas.
